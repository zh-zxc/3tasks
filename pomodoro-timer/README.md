# 番茄钟应用

一个简洁美观的番茄钟Web应用，帮助您提高工作效率和时间管理能力。

## 功能特性

- **核心计时功能**：工作时间、休息时间和长休息时间
- **模式切换**：轻松切换工作、休息和长休息模式
- **自定义时长**：可根据个人需求调整各时间段的时长
- **任务管理**：添加、完成和删除任务列表
- **统计追踪**：记录完成的番茄钟数量和连续完成数
- **本地存储**：所有设置和数据都保存在本地，不会丢失
- **通知提醒**：时间结束时的视觉和声音提醒
- **响应式设计**：适配各种设备屏幕

## 使用方法

1. **开始使用**：
   - 点击"开始"按钮启动计时器
   - 点击"暂停"按钮暂停计时器
   - 点击"重置"按钮重置当前计时

2. **切换模式**：
   - 工作模式：专注工作时间
   - 休息模式：短暂休息时间
   - 长休模式：较长时间的休息

3. **调整时长**：
   - 在设置区域调整工作、休息和长休息的时长

4. **任务管理**：
   - 在任务区域添加待完成的任务
   - 完成任务后勾选复选框
   - 不再需要的任务可以删除

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- 使用localStorage进行数据持久化
- 使用Web Audio API播放提示音
- 使用Notification API显示桌面通知

## 部署到GitHub Pages

此应用已配置为可以直接部署到GitHub Pages：

1. 将代码推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择源码分支（通常是main或master）
4. 应用将在 `https://<username>.github.io/<repository-name>` 上可用

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）。
