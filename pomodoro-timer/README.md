# 番茄钟应用

一个简洁美观的番茄钟Web应用，帮助您提高工作效率。

## 功能特性

- **计时功能**：工作时间（25分钟）、休息时间（5分钟）、长休息时间（15分钟）
- **模式切换**：可在工作、休息、长休模式间自由切换
- **控制按钮**：开始、暂停、重置功能
- **自定义设置**：可调整各时间段时长
- **任务管理**：添加、完成、删除任务
- **统计追踪**：记录完成的番茄钟数量和连续完成数
- **可视化进度**：圆形进度条显示剩余时间
- **通知提醒**：时间结束时的视觉和声音提醒
- **数据持久化**：使用localStorage保存设置和数据

## 技术特点

- 纯HTML/CSS/JavaScript实现
- 响应式设计，适配移动设备
- 使用Canvas绘制圆形进度条
- 集成Web Notifications API
- 使用Web Audio API播放提示音
- 面向对象的JavaScript架构
- 模块化代码结构

## 部署

此应用为纯静态文件，可直接部署到任何Web服务器或GitHub Pages：

1. 将所有文件上传到服务器
2. 确保index.html可直接访问
3. 应用即可正常运行

## 使用方法

1. 选择工作/休息/长休模式
2. 点击"开始"按钮开始计时
3. 可随时暂停或重置计时器
4. 在设置中调整时间长度
5. 在任务列表中添加和管理任务

## 浏览器兼容性

支持所有现代浏览器（Chrome, Firefox, Safari, Edge等）。
